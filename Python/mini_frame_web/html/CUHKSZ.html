    关于我们

    香港中文大学（深圳）是一所经国家教育部批准，按《中外合办大学条例》设立的大学，特色是国际化的氛围、中英并重的教学环境、书院制传统、通识教育、新型交叉学科设置和以学生为本的育人理念。

    书院

    沿袭香港中文大学的书院制传统，目前大学已成立逸夫、学勤、思廷、祥波四所书院。书院制不仅打破了学院和专业的界限，更提供了多元的全人教育环境，充分发挥学生成长潜力。
